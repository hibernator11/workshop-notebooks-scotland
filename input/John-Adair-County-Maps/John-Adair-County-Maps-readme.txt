Title: John Adair - county maps 1680-1720 dataset
Description: This dataset represents the complete descriptive metadata for John Adair - county maps 1680s (http://maps.nls.uk/counties/), a digitised collection of  John Adair's manuscript maps frm his surveys of Scotland in the 1680s. The file format is zip and contains 2 files: 1 plain text file containing the information on this page, 1 XML file of 13 records in MARCXML format (https://www.loc.gov/standards/marcxml/) and 1 XML file in Dublin Core format (https://dublincore.org/).
